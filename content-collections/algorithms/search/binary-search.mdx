<h3 className="text-lg font-semibold">- How it works</h3>

It will halving array until either there's no half left to look at or we find our value.

So we have sorted array of size N. Then we half it. Then we will check if our value is lower, equel of higher then value in the middle. If it is lower then we will get left part of array and now we have a value space that we're looking at that is 1/2N. If we continue halving then we will have value space (length of array) 1/4N, 1/8N ... N/2^K = 1

N = 2^K -> log2N (log of base 2 is equal of N) -> logN

If we have array with 4096 elements we will need 12 times to halving it to get it to 1 (4096, 2028, 1024 ... 1)

log4096 = 12

<h3 className="text-lg font-semibold mt-7">- Big(O)</h3>

It is **O(log N)**.

```ts
export default function bs_list(haystack:number[], needle: number): boolean {
  
  let lo = 0;
  let hi = haystack.length;
  
  do{
    const m = Math.floor(lo + (hi - lo) / 2);
    const v = haystack[m];

    if(v === needle){
      return true;
    }else if(v > needle){
      // left side
      hi = m;
    }else{
      // right side
      lo = m + 1
    }
  }while(lo < hi)

  return false;
}
```
